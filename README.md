## 🚀 Installation

### Windows

```powershell
# 1. Lade die neueste Version herunter
# 2. Führe Trade-Optimizer-AI-Setup.exe aus
# 3. Folge dem Installationsassistenten
# 4. Starte die Anwendung über das Desktop-Icon
```

**Alternativ mit Chocolatey:**
```powershell
choco install trade-optimizer-ai
```

### macOS

```bash
# 1. Lade die .dmg Datei herunter
# 2. Öffne die DMG-Datei
# 3. Ziehe Trade-Optimizer-AI in den Applications-Ordner
# 4. Bei "nicht verifiziertem Entwickler":
sudo spctl --master-disable
# (Nach der Installation wieder aktivieren)
```

### Linux

```bash
# Debian/Ubuntu
wget https://github.com/abalbinali/Trade-Optimizer-AI-2025-h5/releases/latest/download/trade-optimizer-ai.deb
sudo dpkg -i trade-optimizer-ai.deb

# Oder manuell
chmod +x trade-optimizer-ai.AppImage
./trade-optimizer-ai.AppImage
```

## 🎯 Erste Schritte

### 📝 Schritt-für-Schritt-Anleitung

1. **Account erstellen**
   - Starte die App
   - Klicke auf "Registrieren"
   - Bestätige Deine E-Mail

2. **Profil konfigurieren**
   - Wähle Deine Erfahrungsstufe
   - Definiere Dein Risikoprofil (konservativ/moderat/aggressiv)
   - Verbinde Deine Börsen-APIs (optional)

3. **Erste Analyse starten**
   - Klicke auf "Neue Analyse"
   - Wähle die gewünschten Märkte
   - Starte den Scan

4. **Empfehlungen nutzen**
   - Prüfe die KI-Vorschläge
   - Passe Parameter an
   - Setze Trades um (manuell oder automatisch)

## ❓ Fehlerbehebung

### Häufige Probleme & Lösungen

<details>
<summary>🔴 Installation fehlgeschlagen</summary>

- **Antivirus deaktivieren**: Kann die Installation blockieren
- **Als Administrator ausführen**: Rechtsklick > "Als Administrator ausführen"
- **Speicherplatz prüfen**: Mindestens 500 MB frei
</details>

<details>
<summary>🟡 App startet nicht</summary>

- **.NET Framework installieren** (Windows)
- **Terminal-Berechtigungen prüfen** (macOS/Linux)
- **Log-Dateien checken**: `~/.trade-optimizer/logs/`
</details>

<details>
<summary>🟢 Keine Datenaktualisierung</summary>

- **Firewall-Einstellungen prüfen**
- **API-Keys erneuern**
- **Proxy konfigurieren** falls nötig
</details>
